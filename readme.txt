Distributed Systems是一套分布式系统架构
其中包含了数据中心、图片中心、日志中心、以及Util工具集

数据中心
      优点：基于文件持久化存储技术，数据安全、使用简单、容易迁移
      缺点：只能适用于小型系统，数据存放在内存中，对内存有一定要求

图片中心
      优点：图片安全，文件路径使用加密存储
      缺点：获取图片是需通过程序对路径进行解密，降低了效率

日志中心：
      优点：消息队列存储日志，解决高并发
      缺点：海量数据下存在延迟问题，若系统宕机可能会丢失日志

Util：
      优点：内置各种帮助类，完美的解决各种不便
      缺点：操作较基础
